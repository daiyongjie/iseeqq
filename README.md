# iseeqq （通过图片分析操作QQ收发消息）

### 诞生背景

之前一直使用SmartQQ 进行QQ收发消息，最近由于腾讯服务器老是不太稳定，其HTTP服务端经常不可用.

至此诞生出iseeQQ，志在运用图片识别技术来进行QQ收发等操作消息

### 项目介绍

本程序采用 POM 的形式搭建项目， 经量避免了涉及的第三方框架和Jar包，意在减少不必要的引用，增加运行时的负担，

### 注意事项

QQ 运行时需要进行以下操作， 便于程序运行和解析
* QQ前置窗口可以关闭 （可选）
* QQ发送事件 更改为 enter （必须） 
* 建议多彩气泡功能关闭，方便与时间识别 （可选）
* QQ背景建议设置成默，纯色 （可选）
